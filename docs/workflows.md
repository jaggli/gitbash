# Git Workflows — gitbash vs plain git

Gitbash reduces repetitive typing and provides interactive previews and safe cleanup.

- Plain git: ~900 typed characters
- Gitbash: ~300 typed characters

| Action / Grouped Steps                                                                                 | gitbash                                                         | git                                                                                                                                             |
| ------------------------------------------------------------------------------------------------------ | :-------------------------------------------------------------- | :---------------------------------------------------------------------------------------------------------------------------------------------- |
| Checkout base branch                                                                                   | `switch main`                                                   | `git switch main`                                                                                                                               |
| Create feature branch (from latest main)                                                               | `create PROJ-123 add login`                                     | `git switch -c feature/PROJ-123-add-login main`<br>`git push -u origin feature/PROJ-123-add-login`                                              |
| Create a stash (WIP)                                                                                   | `stash "WIP PROJ-123 implement login"`                          | `git stash push -u -m "WIP PROJ-123 implement login"`                                                                                           |
| Create + switch to another branch (ensure main up-to-date)                                             | `create OTHER-456 fix typo`                                     | `git switch main`<br>`git pull origin main`<br>`git switch -c bugfix/OTHER-456-fix-typo main`<br>`git push -u origin bugfix/OTHER-456-fix-typo` |
| Implement changes on other branch, commit & push                                                       | `commit -p "fix(OTHER-456): fix typo"`                          | `git add -A`<br>`git commit -m "fix(OTHER-456): fix typo"`<br>`git push -u origin bugfix/OTHER-456-fix-typo`                                    |
| Switch back to original Jira branch                                                                    | `switch PROJ-123`                                               | `git switch feature/PROJ-123-add-login`                                                                                                         |
| Verify main was up-to-date at creation time<br>Update feature branch by merging current `main` into it | `update`                                                        | `git fetch origin`<br>`git log --oneline HEAD..origin/main`<br>`git pull origin main` (run while on feature branch)                             |
| Review changes (status + diff)                                                                         | `status`                                                        | `git status`<br>`git diff`                                                                                                                      |
| Stage & commit                                                                                         | `status` → stage<br>`commit -p "PROJ-123: implement feature X"` | `git add -A`<br>`git commit -m "PROJ-123: implement feature X"`<br>`git push origin feature/PROJ-123-add-login`                                 |
| Create Pull Request (open compare page)                                                                | `pr`                                                            | Open browser<br>`https://github.com/<owner>/<repo>/compare/feature/PROJ-123-add-login?expand=1`                                                 |
| Merge PR & update local main                                                                           | `switch main`                                                   | `git switch main`<br>`git pull origin main`                                                                                                     |
| Create next branch from latest main                                                                    | `create PROJ-124 implement next thing`                          | `git switch -c feature/PROJ-124-new-thing main`<br>`git push -u origin feature/PROJ-124-new-thing`                                              |
| Clean up merged local branches                                                                         | `cleanup`                                                       | `git fetch --prune origin`<br>`git branch --merged main`<br>`git branch -d <branch>`                                                            |
| List / show stashes                                                                                    | `stashes`                                                       | `git stash list`<br>`git stash show -p stash@{0}`                                                                                               |
| Apply / remove stash                                                                                   | `unstash`                                                       | `git stash apply stash@{0}`<br>`git stash drop stash@{0}`                                                                                       |
| Clean up stashes                                                                                       | `cleanstash`                                                    | `git stash clear`                                                                                                                               |

Notes

- This document compares a full feature workflow step-by-step.
- The "gitbash" column assumes the commands from gitbash are installed globally (no `gitbash` prefix). Example Jira keys: `PROJ-123`, `OTHER-456`.
